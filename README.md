Maths Brain Drill - Scratch Educational Project
A dynamic, interactive mathematics game built in Scratch designed to help young learners improve their mental math speed and accuracy. 
Educational Objectives
•	To teach fundamental arithmetic operations through interactive gamification.
•	To reinforce core programming logic (loops, conditionals, and variables) in a classroom setting.
Project Details
1.	Target Audience: Primary School Learners
2.	Platform: Scratch 3.0
3.	File Included: MATHS BRAIN DRILL.sb3` (Download and open in Scratch to view code blocks and play).
   
Developed as part of an ICT curriculum to empower students through gamified learning.
